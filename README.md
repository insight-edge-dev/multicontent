# 🚀 MultiContent

A modern AI-ready content aggregation platform built with Next.js 14, Prisma, PostgreSQL, and Tailwind CSS.

MultiContent combines:
- 📰 live technology news
- 🎥 curated video discovery
- 🔖 bookmark persistence
- 🔐 authentication
- ⚡ intelligent aggregation
- 🌙 futuristic dark UI

---

# ✨ Features

## 📰 Aggregated News System
- GNews API integration
- Assam Tribune web scraping
- merged news feeds
- category normalization
- resilient aggregation architecture
- caching layer
- article persistence with PostgreSQL

- YouTube-powered video feed
